Plantzr

An accompanying mobile app for the Plantzr web application.

Plant descriptive cards can be created with a picture inserted from the deviceâ€™s camera.
The cards are automatically tagged with the GPS coordinates of the shooting location.

Created with the Ionic framework, it has both an Android and IOS version.

Authentication

In order to be able to post or edit a card, the user must be authenticated.

If authentication was successfull, the OAuth token is stored in local storage. As this information is persisted on disk, the user needs to authenticate only once after installing the app.
